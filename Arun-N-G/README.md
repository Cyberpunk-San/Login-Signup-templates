# 🌐 Advanced Login & Signup Page Template

A **modern**, **fully responsive** authentication page template built with **HTML5** and **Tailwind CSS**.
It features a **glassmorphism-inspired UI**, an **animated gradient background**, and **interactive form toggling** for a seamless user experience.
Perfect for modern web applications, startups, and developer portfolios.

---

## ✨ Features

- **Sleek, Modern UI/UX** – Clean, professional design optimized for engagement.
- **Fully Responsive** – Works beautifully across all devices: desktop, tablet, and mobile.
- **Glassmorphism Effect** – Translucent containers for a sophisticated, layered appearance.
- **Animated Gradient Background** – A dynamic, eye-catching gradient that continuously flows.
- **Interactive Form Toggling** – Instantly switch between Login and Signup without reloading.
- **Social Login Ready** – Circular icon buttons for Google, Facebook, and X (Twitter).
- **Enhanced Input Fields** – Built-in icons and smooth focus animations for better usability.
- **Easy to Customize** – Utility-first design with Tailwind CSS for quick, precise edits.

---

## 🚀 Technologies Used

- **HTML5** – Semantic and accessible structure.
- **Tailwind CSS** – Utility-first responsive styling.
- **JavaScript** – Enables form toggling and interactivity.
- **Google Fonts (Poppins)** – Clean and modern typography.
