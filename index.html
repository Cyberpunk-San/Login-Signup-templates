Contribution Steps

If you have a login or signup template you'd like to add, please follow these steps:

  1. Inside the main directory, create a new folder with a descriptive name (e.g., Modern-Dark-Login).
  2. Add your code:
    Place all of your template's files (index.html, style.css, script.js, etc.) inside the folder you just created.
  3. Add a screenshot:
    • Take a screenshot of your template in action.
    • Save it as a .png file.
    • The filename must exactly match your folder's name (e.g., Modern-Dark-Login.png).
    • Place your screenshot inside the main screenshots folder.
  4. Update the website:
    • Open the main script.js file at the root of the project.
    • Find the templateFolders array at the top of the file.
    • Add the name of your new folder to this list in alphabetical order.
  5. Commit and push your changes:
    git add .
    git commit -m "Add: Your Template Name"
    git push origin feature/your-template-name
  6. Create a Pull Request with a clear title, description, and a screenshot or video demo. We will review it as soon as possible!
